Learn Where to Get Started.

HTML

ウェブページの作成に使用される言語です。 HTMLとCSSを組み合わせることで、 静的なページを作り上げることが出来ます。

jQuery

素敵な動きを手軽に実装できるJavaScriptライブラリです。 アニメーション効果を付けたり、Ajax（エイジャックス）を使って外部ファイルを読み込んだりといろいろなことが出来ます。

Javascript

オープンソースの動的なプログラミング言語で、シンプルさと高い生産性をそなえています。 大きなWebアプリケーションから小さな日常ツールまで、 様々なソフトウェアを作ることが出来ます。

PHP

HTMLだけではページの内容を変えることはできません。PHPはHTMLにプログラムを埋め込み、 それを可能にします。
